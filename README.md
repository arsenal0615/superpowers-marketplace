# Superpowers Marketplace

这是 [arsenal0615/superpowers](https://github.com/arsenal0615/superpowers) 插件的 Claude Code marketplace。

## 安装

在 Claude Code 中执行：

​```bash
/plugin marketplace add arsenal0615/superpowers-marketplace
/plugin install superpowers@qx-superpowers-marketplace
​```

## 卸载

​```bash
/plugin uninstall superpowers@qx-superpowers-marketplace
​```
